# Todo List App

A simple Todo List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Prevent empty task submission
- Local Storage support
- Responsive layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## What I Learned

- DOM Manipulation
- createElement()
- appendChild()
- classList
- addEventListener()
